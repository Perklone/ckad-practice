# Prior knowledge: basic kubectl and K9s usage

User has working experience with Kubernetes in a professional context. Can navigate clusters using K9s, run `kubectl get` commands, and has written manifests with AI assistance. Understands that pods, deployments, and services exist but cannot confidently explain how they relate or write manifests from memory.

## Implications
- Skip "what is Kubernetes" introductions entirely
- Start from the concepts behind what they already use (pods, deployments) rather than introducing them fresh
- Focus on building unassisted manifest-writing ability — the exam has no AI help
- Exam-environment skills (vim, terminal speed, kubectl shortcuts) will be important since they're used to K9s as a crutch
