# Mission: Pass the CKAD Exam

## Why
Employer's L&D team is sponsoring the certification. Kubernetes is already part of daily work (interacting with clusters, reading manifests, using K9s/kubectl), but the underlying concepts aren't solid. Passing CKAD formalizes that knowledge, makes day-to-day work more confident, and strengthens the resume for future career moves.

## Success looks like
- Pass the CKAD exam (score ≥ 66%) within 3-5 months
- Confidently write Kubernetes manifests from scratch without AI assistance
- Explain core k8s concepts (pods, deployments, services, networking, volumes) to a colleague
- Navigate and solve problems quickly in a terminal-only exam environment

## Constraints
- Timeline: 3-5 months, pace depends on available time
- Environment: MacOS (work + personal laptops), Windows PC at home
- killer.sh voucher available but reserved for near-exam-date practice
- Learning happens alongside full-time work

## Out of scope
- CKA (Certified Kubernetes Administrator) — cluster administration, etcd backups, cluster upgrades
- CKS (Certified Kubernetes Security) — advanced security policies
- Cloud-provider-specific k8s (EKS/GKS/AKS specifics beyond what CKAD tests)
